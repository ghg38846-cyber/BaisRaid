# Base Raid (прототип)

2D top-down шутер в духе Hotline Miami + защита базы + волны врагов.

Студенческий pet-проект (Python + Pygame).

## Запуск

```bash
cd survival-base-game
pip install -r requirements.txt
python main.py
```

## Управление

| Клавиша | Действие |
|---------|----------|
| WASD | Движение |
| Мышь | Направление |
| ЛКМ | Стрельба |
| ПКМ | Смена оружия |
| E | Открыть сундук (нужен ключ) |
| 1 / 2 / 3 | Прокачка между волнами (в перерыве) |
| 4–7 | Магазин в перерыве |
| F5 | Сохранить игру |
| C | Продолжить (из меню, если есть save) |
| O | Настройки |
| F11 | Полный экран |
| Enter | Старт / в меню |
| Esc | Меню / выход |

## Геймплей

- Защищайте **базу** в центре карты.
- **4 волны** = 1 уровень, **4 уровня** = победа.
- **Босс** появляется на **4 уровне, 4 волне** (+ рой миньонов).
- **Враги:** обычный, раннер (оранж.), танк (сер.), **стрелок** (фиол.), **рой** (зел.).
- Стрелок стреляет на расстоянии; босс — залп из 5 пуль.
- Сундуки, лут, магазин, перки, турель — как раньше.
- **Сохранение** в `savegame.json` (авто в перерыве + F5).
- **Настройки** в `config.json` (громкость, fullscreen).
