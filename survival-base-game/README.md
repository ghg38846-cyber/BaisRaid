# Base Raid (прототип)

2D top-down шутер в духе Hotline Miami + защита базы + волны врагов.

## Запуск

```bash
cd survival-base-game
pip install -r requirements.txt
python main.py
```

## Управление

| Клавиша | Действие |
|---------|----------|
| WASD | Движение |
| Мышь | Направление |
| ЛКМ | Стрельба |
| E | Открыть сундук (нужен ключ) |
| 1 / 2 / 3 | Прокачка между волнами (только в перерыве) |
| Enter | Старт / в меню после игры |
| Esc | Выход |

## Геймплей

- Защищайте **зелёную базу** в центре карты.
- **Волны** врагов: убили всех → перерыв на лут и прокачку → следующая волна.
- С врагов падают: монеты, ключи, аптечки, патроны.
- **Сундуки** (коричневые) открываются ключом (E рядом).
- **Монеты** в перерыве: `1` — HP, `2` — скорость, `3` — урон.
- Старт с **пистолетом**; винтовка, автомат, дробовик — только из **сундуков**.
- **4 волны** = 1 уровень, **4 уровня** = победа.
- После уровня: **выбор перка** (1/2/3), сундуки **закрываются заново** с tier I–IV.
- Tier сундука = номер уровня (лучше лут).
- **Волны 3–4:** ночь (темнее, враги видят дальше).
- **Враги:** раннер (оранж.), танк (серый), обычный (красный).
- **Турель** у базы — магазин `[7]` или перк.
- **Магазин** между волнами: `[4]` аптечка, `[5]` ключ, `[6]` ремонт базы, `[7]` турель.

## Файлы

- `main.py` — запуск
- `game.py` — цикл, волны, UI, карта
- `objects.py` — игрок, враги, пули, лут, сундуки
- `settings.py` — константы

Графика — цветные фигуры (сначала геймплей, потом спрайты).
